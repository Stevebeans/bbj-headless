## Custom Editor

While making posts on Gutenberg is okay, it's generally a slow system, and I'd also like to have sub writers being able to easily select the show, category, title, featured image, and automatically have any images inserted be automatically centered plus any twitter / instragam embed. I want them to always be consistent. My post editor for basic block posts should feature no more than a basic editor. If I want to go into a sub page and create some gutenberg thing, I can do that myseslf. I want a nice, easy to use, and very customizable post editor that actually has a lot of seo stuff built right in (similar to yoast). A recommended title button that reads the content and generates an SEO title via Claude tokens. One that will be really smart title that will check the sub category and think of that.

Example

-> Live Feed Update Thread

This will automatically create a post like (season) "Big Brother 28 - Live Feed Thread For (date)" While I am not sure spamming Big Brother 28 down my front page is good, it is a frmat that has worked for a long time.

It will also automatically embed a live feed widget that displays the feed updates over the last 24h in a sortable way on the front end
