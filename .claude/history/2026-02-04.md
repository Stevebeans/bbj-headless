# 2026-02-04 — Feed Updates Page Enhancements

## What Was Done

### 1. WordPress Backend Filters Fixed (`HomeRoutes.php`)
- Added `page`, `sort`, `date_range`, `mode`, `search` params to `/bbjd/v1/feed-updates`
- Sort: newest (default, by modified DESC), oldest, highest/lowest (by `total_rating` meta)
- Date range: today, yesterday, week, month, year (uses `date_query` on `post_modified`)
- Mode filter: `_feed_update_mode` meta key (feed/show) — note: mode not stored on existing posts yet
- Search: uses WP_Query `s` parameter
- Removed `no_found_rows => true` for proper `found_posts` count
- Returns `total`, `total_pages`, `page`, `per_page` in response
- `per_page` now allows up to 100 (was capped at 30)
- Page-to-offset conversion handled server-side

### 2. Per-Page Preference Endpoints
- Added `GET/PUT /bbjd/v1/settings/preferences` to HomeRoutes.php
- Stores `bbjd_feed_per_page` in `wp_usermeta`
- Only premium users (supporter/lifetime/admin roles) can update
- Validates 10-100 range
- Frontend API functions added to `src/lib/api/settings.js`: `getPreferences()`, `updatePreferences()`

### 3. Sidebar Layout on Feed Updates Page
- Changed `/feed-updates` from `max-w-4xl mx-auto` to standard two-column layout
- Uses `v2-primary-container` with `flex lg:flex-row` pattern (matches homepage)
- Server-side fetches houseboard data via `getHouseboard()`
- Sidebar includes: Houseboard, SocialFollow, WatchLiveFeeds widgets

### 4. Pill-Style Mode Toggle
- Added prominent rounded-pill buttons above filter bar: All Updates / Feed Updates / Show Updates
- Active state: primary-500 for All/Feed, secondary-500 for Show
- Synced with Type dropdown — changing either updates the other
- Dark mode compatible

### 5. Per-Page Selector & Pagination Polish
- Premium users see per-page dropdown (10/20/30/50/100) in results bar
- Preference loads on mount and auto-saves to backend
- `total_pages` from backend drives pagination correctly
- Non-premium users default to 20 per page (no selector shown)
- Settings page: added "Preferences" section under Premium tab with per-page dropdown

## Files Modified
- `bbj-plugin/src/Api/HomeRoutes.php` — Backend filters + preferences endpoints
- `src/app/feed-updates/page.jsx` — Sidebar layout
