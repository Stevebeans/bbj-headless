### Spoiler-bar refactored to break page-cache cascade
**Why:** Observability showed `/live-feed-updates/[slug]` at **7:1 ISR writes to reads**
(766 / 110, 489 unique paths). Root cause: `SpoilerBarWrapper` did a server-side
fetch with tag `spoiler-bar` — every page rendering the bar (homepage, `/[slug]`,
feed-update detail, player pages, season pages) inherited the tag, so every
player/season status change webhook invalidated all of them.

**Fix:**
- New `src/app/api/spoiler-bar/route.js` — edge-cached (`dynamic = "force-static"`,
  `revalidate = false`) GET handler. Inherits the `spoiler-bar` tag from
  `getCurrentSeasonPlayers` so webhook revalidation still flows through.
- `src/components/spoiler-bar/SpoilerBar.jsx` — now `"use client"`. Fetches
  `/api/spoiler-bar` on mount. **Skeleton-first rendering:** SSR HTML contains
  N placeholder cards with identical dimensions; client fetch swaps real
  cards in. Removes the "bar appears from nothing" flash.
- `src/components/spoiler-bar/SpoilerBarWrapper.jsx` — now a tiny sync SSR
  shell that reserves `min-h-[100px] lg:min-h-[124px]` and mounts the client
  component. **No more `spoiler-bar` tag on any page cache.**
- `SKELETON_COUNT = 17` hardcoded for BB27. **Bump back to 16 when BB27 ends.**
- User removed `animate-pulse` from the skeleton (preferred no throb).
- Webhook flow still works: `revalidateTag("spoiler-bar")` invalidates only
  the `/api/spoiler-bar` route output; pages stay cached.

### BB Time clock bug fix
**Why:** Header showed wrong time ("-4h from EST"). Code looked correct
(`timeZone: "America/Los_Angeles"` is right and auto-handles DST). Real bug:
`new Date()` was computed once at render and never updated — frozen at the
ISR-regeneration moment, then frozen again at user-page-load moment with no
ticker.

**Fix:**
- `src/hooks/useBbTime.js` — new shared hook. `useEffect` overwrites stale
  SSR value on hydration, then `setTimeout` syncs to next minute boundary,
  then `setInterval` ticks every 60s.
- `src/components/layout/Header.jsx` — replaced inline call with `useBbTime()`.
- `src/components/layout/MobileMenu.jsx` — same.

overlay. Toggleable at: DevTools → ⋮ → More tools → Rendering. Useful while
fixing CLS, turn off for clean visual inspection.
